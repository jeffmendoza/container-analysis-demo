# Container Analysis Webhook

## Usage

```shell
$ ./container-analysis-webhook -h
Usage of ./container-analysis-webhook:
  -sev-thresh string
    	Severity Threshold: LOW, MEDIUM, HIGH, or CRITICAL (default "HIGH")
  -tls-cert string
    	TLS certificate file. (default "/etc/admission-controller/tls/cert.pem")
  -tls-key string
    	TLS key file. (default "/etc/admission-controller/tls/key.pem")
```
